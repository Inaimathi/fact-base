# fact-base
#### Simple implementation of fact-base data storage for Common Lisp

### Requirements

`:fact-base` depends on `alexandria`, `anaphora`, `local-time`, `optima` and `cl-fad`. All of them installable via `quicklisp`.

### Usage

##### `fact-base`
##### `:matching?`
##### `:file-name`
##### `:next-id!`
##### `:select`
##### `:insert!`
##### `:multi-insert!`
##### `:delete!`
##### `:project!`
##### `:project`
##### `:write!`
##### `:update!`
##### `:load!`
